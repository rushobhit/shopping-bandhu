Shopping Bandhu

Self-billing machine concept combining embedded hardware with a lightweight web interface.

About the Project

Shopping Bandhu is a self-billing machine concept developed as an IoT-based product solution that combines embedded hardware with a simple web interface to improve the customer billing experience.

The project was built around the idea of creating a smoother and more practical retail checkout flow by connecting hardware actions with a lightweight digital interface.

Hardware

NodeMCU ESP8266, NFC tags, weight sensors, and servo motors were used to build the physical and sensing part of the system.

Software

The system was programmed using Arduino IDE, while the web interface was built using HTML, CSS, and JavaScript to display cart details and billing flow.

Objective

The main goal was to connect hardware and software into one working self-billing experience that feels simple, fast, and practical.

Outcome

The project strengthened understanding of IoT product development, retail automation concepts, system integration, and hands-on technical problem solving.
